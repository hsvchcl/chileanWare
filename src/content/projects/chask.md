## Sobre Chask

Chask es una herramienta de línea de comandos escrita en Go que facilita la gestión de proyectos y la automatización de tareas repetitivas en el desarrollo de software.

### Características principales

- **Templates de proyecto**: Bootstrapping rápido con templates configurables.
- **Pipeline de tareas**: Define y ejecuta pipelines complejos con una sintaxis simple.
- **Integración CI/CD**: Generación automática de configuraciones para GitHub Actions, GitLab CI, etc.
- **Multiplataforma**: Disponible para macOS, Linux y Windows.

### Instalación

```bash
brew install chask
# o
go install github.com/matias-cx/chask@latest
```
