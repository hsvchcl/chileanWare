## Sobre Fintual

Fintual es una administradora general de fondos chilena que permite invertir de manera simple, transparente y con bajas comisiones. Fundada en 2018, ha revolucionado el mercado financiero chileno.

### Características principales

- **Inversión automatizada**: Algoritmos que diversifican tu portafolio según tu perfil de riesgo.
- **Comisiones bajas**: Las más competitivas del mercado chileno.
- **Interfaz moderna**: UX/UI diseñada para hacer la inversión accesible a todos.
- **Educación financiera**: Blog y recursos educativos de primer nivel.

### Stack tecnológico

- Ruby on Rails para el backend
- React para el frontend
- PostgreSQL como base de datos
- Infraestructura en AWS
