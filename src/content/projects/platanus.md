## Sobre Platanus

Platanus es un estudio de desarrollo de software y venture builder fundado en Santiago de Chile. Además de crear startups, son reconocidos por su fuerte compromiso con el open source y la comunidad de desarrolladores chilena.

### Herramientas Open Source destacadas

- `activeadmin_addons` — Extensiones para ActiveAdmin en Rails
- `potassium` — Generador de proyectos Rails con las mejores prácticas de Platanus
- `power_types` — Patrón Services, Commands, Values y Utils para Rails
- `bank_api` — Scraper de bancos chilenos
- `lita` — Bot framework para equipos
- `vue-coin` — Componentes Vue para proyectos Platanus

### Contribuciones a la comunidad

- **Blog técnico**: Publicaciones sobre Ruby, Rails, Vue.js y buenas prácticas de desarrollo.
- **Open Source**: Más de 100 repositorios públicos en GitHub.
- **Venture Building**: Han creado startups como Fintual (que luego se independizó).

### Stack tecnológico

- Ruby on Rails para backend
- Vue.js para frontend
- PostgreSQL como base de datos
- Heroku y AWS para infraestructura
