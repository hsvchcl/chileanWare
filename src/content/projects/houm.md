## Sobre Houm

Houm es una startup proptech chilena que busca transformar la experiencia de arrendar y vender propiedades en Latinoamérica mediante tecnología.

### Características principales

- **Visitas virtuales**: Tours 3D y videollamadas para conocer propiedades remotamente.
- **Contratos digitales**: Firma electrónica avanzada para contratos de arriendo.
- **Pagos automatizados**: Cobro de arriendo automático y dashboard para propietarios.
- **App móvil**: Disponible para iOS y Android.

### Stack tecnológico

- React Native para la app móvil
- TypeScript en todo el stack
- Node.js + NestJS para microservicios
- PostgreSQL y Redis
