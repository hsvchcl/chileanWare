## Sobre Buda.com

Buda.com (anteriormente SurBTC) es un exchange de criptomonedas fundado en Santiago de Chile en 2015. Opera en Chile, Colombia, Perú y Argentina.

### Características principales

- **Multi-criptomoneda**: Soporte para Bitcoin, Ethereum, Litecoin y más.
- **API robusta**: API REST completa para integrar trading automatizado.
- **Seguridad**: Almacenamiento en frío y autenticación de dos factores.
- **Open Source**: Múltiples librerías de código abierto publicadas.

### Contribuciones Open Source

- `trading-bots` - Framework para bots de trading
- `ruby-api-client` - Cliente Ruby para la API de Buda
- Múltiples SDKs en distintos lenguajes
